# 📒 NoteZy - Smart Notes & Task Manager App (iOS)

NoteZy is a simple and efficient iOS application that helps users manage their **Notes, Reminders, To-Do Lists, and Calendar Events** in one place. Designed with a clean UI and smooth user experience, the app improves productivity and daily organization.

---

## 🚀 Features

### 📝 Notes
- Create, edit, and delete notes
- Store important ideas and information
- Clean and minimal note interface

### ⏰ Reminders
- Set reminders for important tasks
- Never miss deadlines or activities
- Easy scheduling and alerts

### ✅ To-Do List
- Create task lists
- Mark tasks as completed
- Track daily productivity

### 📅 Calendar Events
- Add and manage events
- View scheduled activities
- Organize your day efficiently
